# Marketplace Supplier Management

Sistema fullstack para **gestão de fornecedores, produtos e cotações**, desenvolvido com **React, TypeScript, Node.js e Prisma**.

O projeto permite que administradores gerenciem fornecedores e produtos, enquanto fornecedores podem cadastrar produtos e participar de **cotações de compra**.

---

# 🚀 Tecnologias

## Frontend
- React
- TypeScript
- React Router
- Context API
- Axios
- SCSS

## Backend
- Node.js
- Express
- TypeScript
- Prisma ORM
- JWT Authentication

## Banco de dados
- PostgreSQL (via Prisma)

---

# 📦 Funcionalidades

### 🔐 Autenticação
- Login
- Cadastro
- Controle de acesso por **roles**
  - ADMIN
  - SUPPLIER

---

### 👨‍💼 Administração

Administrador pode:

- Aprovar fornecedores
- Visualizar fornecedores pendentes
- Gerenciar produtos
- Criar e gerenciar cotações

---

### 🏪 Fornecedores

Fornecedores podem:

- Criar produtos
- Editar produtos
- Visualizar suas cotações
- Participar de processos de cotação

---

### 📦 Produtos

Funções disponíveis:

- Criar produto
- Listar produtos
- Atualizar produto
- Deletar produto
